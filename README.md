# 杀人游戏总览
    游戏名称：死亡面具（《电锯惊魂1》）

　　参加者：阿曼达（女性）

　　入选原因：吸食毒品

　　游戏时间：1分钟

　　游戏设置：阿曼达头戴一顶设计精密的“死亡面具”。该面具犹如一只反向运作的“捕猎夹”，两片夹片分别固定在她的上下唇，呈闭合状。倘若无法在规定时间内卸下面具，那么两片夹片就会大力弹开，导致游戏者因上下颚分离而死。而唯一能解开面具的钥匙，则被缝在了阿曼达室友的体内，必须开膛破肚才能取出。

　　游戏结果：阿曼达成为迄今为止唯一一位未作弊而通关的幸存者。为了求生，她在疯狂和绝望中杀害了室友，并于最后一秒摘下“死亡面具”。

　　Jigsaw语录：许多人活之有愧，而你不再是其中一员。

　　

 

    游戏名称：刺颈电钻（《电锯惊魂1》）

　　参加者：杰夫、探员申（男性）

　　入选原因：不详

　　游戏时间：20秒

　　游戏设置：杰夫被锁在椅子上无法动弹，脖子两旁各有一把电钻正逐步逼近。探员申必须在20秒内从一堆钥匙中找出匹配的一把，来解开杰夫脖子上的锁套。

　　游戏结果：由于时间紧迫，申没有试完所有的钥匙，而是选择用枪打断了电钻。这一作弊行为使他在之后的追捕中受到了惩罚。

　　

 

    游戏名称：烛火焚身（《电锯惊魂1》）

　　参加者：马克（男性）

　　入选原因：佯装生病，懒于工作

　　游戏设置：马克必须在毒发之前，从墙上密密麻麻的数字中找到密码，打开暗室中央的保险箱，取出解药。但要小心的的是，赤身**的他不仅被注入了慢性毒素，周身也涂满易燃物，而且只有一种照明工具可供选择，那就是——蜡烛。另外，地上还布满玻璃碎片。

　　游戏结果：从烧焦的尸体分析，马克不是毒发身亡后，享受了火葬的待遇；就是在火中了却了残生。

　　

 

    游戏名称：荆棘迷宫（《电锯惊魂1》）

　　参加者：保罗（男性）

　　入选原因：曾割腕自杀

　　游戏时间：2小时

　　游戏设置：保罗被困在地下室的一个遍布铁丝网的牢笼中。他有两个小时的时间从带刺的“铁荆棘丛”中杀出一条血路，否则地下室的门就将永久关闭。然而，为了出去，仅仅身着内裤的他，也将被“铁荆棘”割得遍体鳞伤。

　　游戏结果：保罗用尽全力、流尽鲜血也未能走出“荆棘迷宫”。虽然他离出口仅几步之遥，可仍因失血过多而命丧铁笼。

　　Jigsaw语录：“你的割腕自杀究竟是真的对生已绝望，还是想借此博取大家注意？现在，我给你一个机会来证明自己。不过，讽刺的是，如果你想死，只要留在那儿即可；如果想生，就必须再次割伤自己。保罗，为了活命，你愿意流多少血？受多少伤？”

　　

 

    游戏名称：散弹齐射（《电锯惊魂1》）

　　参加者：探员申（男性）

　　游戏设置：四把散弹枪被悬挂在门廊上，一根钢丝绳将它们的扳机连在一起。当申踩中下方的钢丝绳时，头顶的散弹枪就将四枪齐发。从严格意义上说，这个只能算是一个安保装置，而算不上是Jigsaw“创意游戏”的一种。但在一点上，这一陷阱和Jigsaw的死亡游戏保持一致：被害者因自己的失误而殒命。

　　游戏结果：由于擒凶心切，探员申误触陷阱，为他之前作弊付出了生命的代价。

　　

 

    游戏名称：监视浴室

　　游戏者：医院的一名护工泽普（对他影片没有详细的介绍，但据推测也应该是个瘾君子）

　　游戏设置：泽普体内被约翰注入慢性毒药，他的游戏规则只需要静静的做在监视器前面监视浴室内两个游戏者（医生戈登和偷窥狂亚当）的动静，并在六点到来之时杀掉戈登医生的老婆即可，完成任务便可拿到解药。

　　游戏结果：计划没有变化快，在绑架戈登妻女时，最后同戈登妻子搏斗过程中受伤，这时他必须跑到浴室杀掉戈登医生才可挽回游戏败局，但在浴室里被亚当用马桶盖拍死，至此游戏失败。

　　

 

    游戏名称：浴室游戏，（第一部的主线游戏）

　　游戏者：背着妻子在外偷情的医生戈登，偷窥狂/私人侦探亚当

　　游戏设置：一个肮脏而简陋的浴室，一边是暴躁的亚当，一边是冷静的医生，俩人的一只腿都被铁锁扣住，并各配有一把锯子，俩人按照收音机里的一些提示，逐步的找到游戏线索，规则便是医生在6点到来之前杀掉亚当（提示告诉他可以用一个盒子里有毒的香烟去毒死亚当），否则中间死尸血液中的毒素便会蔓延，铁门永远锁上，成为俩人的墓地。

　　游戏结果：戈登在最后被妻子的一个电话逼得丧失理智，时间已到，他没有忍心杀掉亚当，并残忍的用锯子锯断自己的脚，冷静的开枪射杀了亚当的肩膀，自己爬了出去，勉强的算通过游戏，亚当则在一开始就把浴池里的钥匙被自己冲走，并把来打算杀戈登的护工泽普用马桶盖砸死，自己也最终被囚禁在此（第三部介绍被阿曼达捂死）

　　ps：实际亚当算是游戏最终的胜利者，但无奈自己把钥匙冲走，戈登医生则用了另外一种非正规的方法通过了游戏。

　　

 

    游戏名称：死亡面具（《电锯惊魂2》）

　　游戏者：迈克（马修警官的线人，也被老头说成所谓的生活中的小人，告密者）

　　游戏设置：迈克所佩戴的面具前后可以闭合，并且内部布满了钢针，一旦闭合整个头就会被扎透，唯一可以开启面具的钥匙被镶在了他的右眼里，在有限的一分钟里，要想活命，只有损失自己的一只眼睛，亲手用刀将钥匙挖出来。

　　游戏结果：多次把刀片触到眼睛边缘的麦克最终还是没能下得了手，时间慢慢的流逝，麦克保住了眼睛，但面具闭合，损失了生命。

　　

 

    游戏名称：聊天游戏（《电锯惊魂2》）

　　游戏者：门主约翰，马修警官（曾经为了自己的名誉而捏造了多起假证）

　　游戏设置：马修只需安静的坐在约翰对面和他聊两个小时即可，什么也不用做。但马修的儿子丹尼却已经被约翰抓住，和其他几个曾经被他父亲冤枉入狱的瘾君子关在一起，而门主保证只要马修能耐得下性子和他谈两个钟头的话，他保证丹尼会平安无恙

　　游戏结果：眼看着密室中的人一个个死去，马修感到死亡渐渐的逼向了他的儿子，再加上FBI看到了老头搜集的一些关于马修的假证，于是马修终于暴跳如雷，将其一顿痛打之后叫带他去密室，最终中了约翰圈套，被阿曼达囚禁在第一部的浴室里。

　　

 

    游戏名称：地狱炼炉（属于主线游戏中的一个过渡游戏）（《电锯惊魂2》）

　　游戏者：奥比（充当第一部泽普角色，约翰的傀儡，帮助约翰绑架了其他人并最终一同被困在此）

　　游戏设置：在晕倒醒来后，奥比与其他几人一同被困在一密室里，并且空气有致命的毒性气体在不断的蔓延，2小时过后就会要了所有人的命，他们不断的寻找线索，并进行了第一个个人游戏，壁炉里有两支解药，按照录音机的提示，一只是留给自己的，另一只可给其他人，‘若你身处地狱，只有魔鬼相助’，

　　游戏结果：奥比最终进去没有再出来，门被锁紧，其他人将其救出后已经被烧死，一只手伸出，拿出的针筒也全部融化，

　　PS：‘身处地狱‘指的是在火炉里，‘魔鬼相助‘指的是转动魔鬼头像标明的水龙头

　　

 

    游戏名称：针坑游戏（属于主线游戏中的一个过渡游戏）（《电锯惊魂2》）

　　游戏者：丹泽尔（一个以自我为中心，但生存力极强的毒贩）阿曼达

　　游戏设置：依然是为了获得解药祛除体内的毒素，奥比死后他们进行了第二个个人游戏，解药就在丹泽尔旁边的门里，但钥匙却在茫茫的针海里，想要解药只有在有限的时间里去找到钥匙。

　　游戏结果：丹泽尔只顾及自己，残忍的将阿曼达扔进针坑里去找钥匙，阿曼达歇斯底里的大叫，最终把钥匙找了出来，丹泽尔去开门的那一刻时间也到了，解药没有拿到。

　　PS：这个游戏虽然失败，但是没有人丧命

　　

 

    游戏名称：锁链游戏（《电锯惊魂3》）

　　游戏者：大部分时间在监狱度过的特洛伊

　　游戏设置：全身被铁链穿透并拴住，屋内装有定时炸弹，时限一分钟，唯一的逃生办法就是死死挣开铁链走出门，要想生还付出的代价就是肉体上的缺失，

　　游戏结果：特洛伊拼尽全力只铮断了几条链子，没有在规定的时间内逃脱最终被炸死

　　PS：此游戏并非门主约翰设计，为门徒霍夫曼所做，门被焊死，就算挣开铁链也难逃一死，根本没有生还希望

　　

 

    游戏名称：死亡天使游戏（《电锯惊魂3》）

　　游戏者：凯利警官（马修的搭档）

　　游戏设置：整个天使游戏的装置钩住了凯利的胸膛，在她面前的酸性溶液内有解开装置的钥匙，时限一分钟，要想不被扯成两半就只有牺牲自己的手去拿钥匙。

　　游戏结果：凯利将手把钥匙拿出来，手几斤腐烂，但钥匙并没有打开身上的装置，时间到时整个胸膛被撕开。

　　PS：此游戏为门徒阿曼达设计，没有给游戏者一丝生存机会

　　

 

    游戏名称：解冻游戏（主线游戏中的过渡游戏）（《电锯惊魂3》）

　　游戏者：杰夫（在车祸中失去了儿子，过去三年一直生活的绝望和悔恨中），丹尼卡斯科特（女，唯一杰夫车祸的目击证人，但拒绝出庭作证）

　　游戏设置：斯科特被吊在冷冻密室里，周围的冷水开关开启，不断的让她的身体‘雪上加霜’，杰夫手中有解锁的钥匙，宽恕还是不原谅由杰夫决定。

　　游戏结果：杰夫犹豫不决，对8岁儿子的丧生无法容忍眼前的斯科特，眼看着她的呼吸越来越微弱，杰夫在最后一刻选择了解锁，但最终还是晚了一步，斯科特被活活冻死。

　

 

    游戏名称：死猪阉人（第三部主线游戏的过渡游戏）（《电锯惊魂3》）

　　游戏者：杰夫，法官（曾经只给杰夫儿子车祸肇事者判了轻刑）

　　游戏设置：法官所躺的地方是一个大坑，上方的机关开启，一只只死猪不断的掉下，经过搅拌机搅拌后掉进坑内，浇在法官的身上，在旁边的焚化炉里装着杰夫儿子的一些玩具，里边藏有钥匙，只能由杰夫按动焚化炉开关才可获得钥匙，宽恕还是不原谅依然由杰夫决定。

　　游戏结果：看着儿子的玩具，杰夫又想起了自己的儿子，他无法容忍这个法官的作为，但法官最后说了一句‘我也有个儿子’，出于对父亲的责任感同身受，杰夫最终选择了原谅，法官获救。

 

 

    游戏名称：人体扭转机（第三部主线游戏里的过渡游戏）（《电锯惊魂3》）

　　游戏者：杰夫，法官，蒂莫西杨（肇事的年轻司机，医学院学生）

　　游戏设置：杨的全身被固定在人体扭扭机器上，机关开启，齿轮一运作，机器会带着人体像变形金刚般扭转，机关的钥匙在旁边的玻璃容器内，并且在其旁边配有一把枪，但拿钥匙必须要付出挨一颗子弹的代价，宽恕还是不原谅，由杰夫做决定。

　　游戏结果：杰夫选择原谅，但耍小聪明，以为把钥匙从枪的后方扯下来就行，但扳机的原理是向反方向运作，正好将在扭扭机旁边的法官爆头，黑人杨也被扭碎，游戏失败。

　　PS：在三个过渡游戏里，杰夫对曾经的仇人都选择了原谅，但因为种种的原因，最终结果没有救活一人。

 

 

    游戏名称：阿曼达最终入门游戏（《电锯惊魂3》）

　　游戏者：阿曼达，门主约翰，莲恩（主线游戏者杰夫的前妻，被门主抓来为其治病，因为失去儿子而对生活不思进取，有一情夫），杰夫

　　游戏设置：在阿曼达成功摆脱死亡头盔游戏后，采用了割腕的方式拜约翰为师加入电锯门，但阿曼达始终没有领略电锯精神的人性内涵，不给游戏者一点生存的希望（凯利的死亡天使游戏，亚当活了下来但被她捂死），基于此点，约翰决定对他进行最后一次考验，为了保证游戏的成功几率，约翰没有告诉她莲恩是杰夫的妻子，阿曼达的规则只要在最后杰夫游戏完成时候不杀莲恩便可通过最终考验。

　　游戏结果：莲恩在给门主约翰做开颅手术时约翰产生了幻觉，误把她当成自己的前妻吉尔，并向其示爱，这些都被阿曼达看在眼里，由此心生妒忌，怨恨（门徒霍夫曼也留下一封信威胁阿曼达要杀掉莲恩，否则就把门主妻子流产的经过告诉约翰），纠结于此的阿曼达最终开枪将莲恩杀死，而自己也被刚刚通关闯进门主屋内的杰夫杀掉，约翰在最后流下了无奈的眼泪，最有希望接替自己电锯门事业的得意弟子没有通过最终考验，死在此地。

 

 

    游戏名称：项圈游戏（《电锯惊魂3》）

　　游戏者：莲恩（杰夫妻子，儿子车祸去世后对生活不思进取，有一情夫，最好的外科大夫之一），杰夫，约翰

　　游戏设置：阿曼达奉命将莲恩带到奄奄一息的约翰面前，并戴上一种特制的项圈，项圈与约翰的心率监测器相互感应，如果约翰的心跳停止，项圈也会爆炸，莲恩只需保证在丈夫杰夫通关前让约翰活下来（或者说让约翰活的更长）就可以，而她并不知道自己的丈夫正在那边做游戏。

　　游戏结果：由杰夫的下个游戏最终决定

 

 

    游戏名称：最终选择（第三部的主线游戏）（《电锯惊魂3》）

　　游戏者：莲恩，杰夫，约翰

　　游戏设置：杰夫并不知道自己妻子的项圈和约翰的心率监测器相连，而站在约翰的面前，他的桌子上有许多可以让人毙命的工具(锯子，手术刀），面对毫无还手之力的约翰，是宽恕还是不原谅，由杰夫决定

　　游戏结果：面对眼前的这个竖锯杀人狂，杰夫最终没有选择原谅，用电锯将约翰割喉致死，由此莲恩的项圈爆炸，被爆头，而约翰也留了后手，没有告诉他的女儿也被囚禁在一个空气稀薄的密室里，所有门被锁死，游戏失败的杰夫只留下一个人生存在悔恨与自责中。





    游戏名称：陵墓嘴眼游戏，（从第二部开始开头都会有个孤立游戏）（《电锯惊魂4》）
